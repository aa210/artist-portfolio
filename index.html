WELCOME TO MY PORTFOLIO


My name is Ade and I am a web designer who helps clients develop stronger relationships with their audience. I achieve this by building clean, user-friendly websites and efficient digital marketing campaigns. My focus is on using the most effective tools to help clients reach their business goals!


Services I Offer

Web Design + Development

Whether you're looking to build a new website or redesign, I can build for you a functional, modern, mobile-responsive website. SEO will be included so the right customers find you in searches.

Marketing + Branding

Digital marketing is the perfect way to grow your audience and boost sales. I'll drive consistent awareness to your brand with tools such as Instagram, blogging, and email marketing so you can focus on operating your business.

Maintenance + Updates

I provide website hosting to web design clients. Clients who choose a monthly maintenance package are guaranteed free hours of design revisions and updates that keeps their website up-to-date and secure. A website ranks higher in searches if it is regularly updated.

FrontEnd Design and Development

SCLD Download

Design/ FrontEnd Development
I was excited to test out Soundcloud’s API by creating an app that generates a random track (under 10 minutes long to avoid long DJ mixes) that can be downloaded based on the genre the user clicks. Tools used: Sass, JavaScript, HTML, BootStrap, CSS3.


Twitch.TV Feed

Design/ FrontEnd Development
This displays the status of selected Twitch.tv accounts using the Twitch.tv API. The user can navigate between users online and offline. I would like to take this concept further (maybe for another project) by allowing the user to input usernames of their choosing to be added to the feed (only if the username exists). Tools Used: jQuery, JSON, HTML, BootStrap, CSS3.

Wikipedia Viewer

Design/ FrontEnd Development
Search the entire Wikipedia Database using this tool. I implemented the MediaWiki API for this project. Begin typing into the search box and watch the results appear, in real time, underneath.
